# 实时翻译 Chrome 扩展

这是一个 Manifest V3 谷歌浏览器扩展，会把网页中的英文文本翻译成中文，并显示在原文下方、左侧或右侧。

## 安装

1. 打开 Chrome，进入 `chrome://extensions/`。
2. 打开右上角的“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择本目录。

## 使用

- 点击浏览器工具栏中的“实时翻译”，可以开启/关闭翻译、选择译文位置。
- 点击“翻译当前页”会扫描当前网页中的英文段落、标题、列表项和表格文本。
- 默认会自动翻译新页面中的可见英文；取消勾选“打开网页后自动翻译可见英文”后，可改为手动翻译。
- 选中网页里的英文，也可以通过右键菜单“实时翻译选中的英文”查看弹出译文。

## 说明

当前版本使用 Google Translate 的公共 Web 请求接口进行翻译，需要浏览器可以访问 `translate.googleapis.com`。这个接口适合个人小工具使用；如果要长期稳定发布到 Chrome Web Store，建议改接正式翻译 API。
